> v2ray installer

## Install

```bash
$ BINDIR=/usr/local/bin bash <(curl -L -s https://raw.githubusercontent.com/zcong1993/v-install/master/install.sh)
```

## Setup v2ray

```bash
$ v-install -install
```

## Update config

```bash
$ v-install -update
```

## Show config

```bash
$ v-install
```
